dissolveAPI docs

Introduction

dissolveAPI is a restAPI that provides many useless things.
dissolveAPI has two main features, mathmatical constants & programming memes.

Base URL

type url
the very base URL http://dissolve.kro.kr:8080/api
constant API http://dissolve.kro.kr:8080/api/const
meme API http://dissolve.kro.kr:8080/api/meme

Request Method

dissolveAPI only handles GET requests.

Constant API

this API provides value of various mathematical constants.

Query Parameters

Parameter name Description type Is Required Default Value
type name of the constant that is requested a string that is written at the Constant Name section of Constant list TRUE None, Required
start the starting digit after the decimal point. if 0, the response includes number before the integer FALSE 0, give value from before the decimal point.
end the ending digit after the decimal point. FALSE start, give only one value.
